◆ do_CWC()

void Cwclass::do_CWC ( const vector< real_prec > &  delta)

Performs the Cosmic-Web classification.

Performs the Cosmic-Web classificatio by solving Poisson equation using FFTW, identifying CW using the eigenvalues of the tidal tensor and classifying the different CW given a threshold Bam::lambdath

Parameters
deltaOverdensity field of the dark matter distribution interpolated on a grid with size Bam::NGRID
Returns
This function loads the container Bam::CWClass of size Bam::NGRID specifying in each cell the type of CW with 1, 2, 3, 4, for knots, filaments, sheets and voids, respectively.

◆ get_Tclassification()

int Cwclass::get_Tclassification ( int  i)

This function provides the index identifying the T-CWT classification as requiested in the parameter file, for each cell, and according to the CWClass array. Example, if requested 1 2 34, the allowed indices are 0,1,2.

Parameters
iindex (row-major order, i<Bam::NGRID) identifying the cell. Example:
for(ULONG ig = 0; ig< NOBJS ; ++ig) // Loop over tracers
{
int I_CWT=0;
ULONG ID= tracer.Halo[ig].GridID; // Get the ID of tracers [0,NOBJS-1]. tracer is type Catalog
I_CWT=wclass.get_Tclassification(ID);// wclass is type Cwclass
}
Returns
Container (size NGRID) with the index of classifiction (according to input par file) for each cell

◆ CWClass

vector<int> Cwclass::CWClass

Container to allocate the cosmic web classification for each cell.

Example: for knots

for (ULONG index=0;index<NGRID;++index)
if (lambda1[index]> lambdath && lambda2[index]> lambdath && lambda3[index]>lambdath) // lambdath = Threshold
CWClass[index]=I_KNOT; // I_KNOT = 1 Preproc definition
