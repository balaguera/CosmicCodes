◆ k_bin_step

real_prec PowerSpectrumF::k_bin_step
private

Type of linear binning in Fourier space: 0.5 or 1. for 0.5, the bins are k_i= (i+0.5)*Delta. In this case the center of the first bin is at the mid point between the zero mode and the fundamental mode if Delta=delta. for 1, the bins are k_i= (i+1)*Delta. In this case the center of the the first bin is at the fundamental mode, if Delta=delta.

◆ sys_of_coord_g

int PowerSpectrumF::sys_of_coord_g
private

coordinate system of the object catalogue

options:

0 → positions are given in Cartesian coordinates (X, Y, Z)

1 → positions are given in equatorial coordinates (RA, Dec, r)

2 → positions are given in pseudo-equatorial coordinates (RA, Dec, z)

3 → positions are given in pseudo-equatorial coordinates (RA, Dec, z)

Note
with option 2, the code uses the set of cosmological parameters to transform redshift z to comoving distance; with option 3, the redshift is directly used as radial coordinate
