Analyze the density fields read by read_bam_files()

This function redefines the values of delta_X_min etc in case it is requeitsd in parameter file. Else, input values in parameter file are used.

Compute the abundance with respect to a X property from the refernece catalog, conditional to the properties of teh DM density field.

This is accomplished by constructing an histogram in the different variables. This is an alternative to the mass-assignmetn approach given by the the calculation of B(M|{theta}), where M is the total mass of tracers in cell. Here instead we compute n(Mh|{}) to assing halo masses (i., e, on an object by object basis). This approach works better. DOES NOT USE THE INFORMATION FROM V-CLASS matter prperties and Vmax

Returns
This function loads the class member container Bam::CWT_X_Y_hist
